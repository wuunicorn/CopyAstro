# 复制星座 / Copy Constellation

## 应用简介 / App Introduction

**复制星座 (Copy Constellation)** 是一款专为占星爱好者、职业占星师以及大模型（LLM）分析设计的专业占星工具。

在 AI 时代，我们不仅提供精准的星盘计算，更致力于将复杂的星象数据转化为结构化、清晰的文字，让 ChatGPT、Claude 等大语言模型能够深度理解并分析您的星盘细节。

**Copy Constellation** is a professional astrology tool designed for enthusiasts, professional astrologers, and Large Language Model (LLM) analysis. In the AI era, we provide not only precise chart calculations but also structured text output optimized for AI like ChatGPT and Claude to deeply understand and analyze your astrological details.

## 核心功能 / Core Features

### 1. 专业星盘体系 / Professional Chart Systems
- **精准本命盘 (Natal Chart)**: 基于 Swiss Ephemeris 引擎，提供极高精度的行星、宫位及相位计算。
- **动态行运 (Transits)**: 实时监测当前天象对个人星盘的影响。
- **深度合盘 (Synastry & Composite)**: 提供比较盘、组合盘及龙头图（Draconic Chart），全方位解析人际关系。
- **高级推运 (Forecasting)**: 支持日返盘、月返盘、次限推运、日弧推进等专业推运技术。

### 2. AI 赋能分析 / AI-Powered Analysis
- **一键复制 (One-Click Copy)**: 将复杂的星盘、相位及推运数据一键导出为 LLM 友好的文本格式。
- **结构化输出**: 自动整理行星落座、宫位描述及相位夹角，消除 AI 分析时的幻觉。

### 3. 高级交互体验 / Advanced UX
- **实时时间微调**: 交互式调整出生时间，即刻观察相位和宫位漂移，适合生时校正。
- **智能档案管理**: 支持拼音快捷搜索、置顶常用档案，轻松管理成百上千的占星资料。
- **完全离线运行**: 所有计算均在本地完成，保护隐私的同时确保随时随地可用。

## 技术实现 / Technical Implementation

- **SwiftUI**: 现代化的 iOS 原生交互设计。
- **Swiss Ephemeris**: 全球公认的高精度天文历法计算库。
- **iOS 17.0+**: 适配最新的 Apple 设计规范和隐私标准。
