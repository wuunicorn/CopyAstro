# 复制星座 / Copy Constellation

## 应用简介 / App Introduction

复制星座 是一款专业的星座信息复制工具，核心功能是将星座信息转换为可以让大模型分析的文字格式。我们致力于为用户提供便捷的星座信息复制服务，让大模型能够更好地理解和分析星座相关内容。

Copy Constellation is a professional constellation information copying tool. Its core function is to convert constellation information into text format that can be analyzed by large language models. We are committed to providing users with convenient constellation information copying services, enabling large language models to better understand and analyze constellation-related content.
