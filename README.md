# 星座分析助手

## 应用简介

星座分析助手 是一款专业的西方占星学分析应用，集成了星座查询、星盘分析和运势预测等核心功能。我们致力于将西方占星学与现代科技相结合，为用户提供准确、便捷的星座分析服务。

## 主要功能

### 1. 星座信息查询
- 查询基本星座信息
- 显示星座特征
- 提供星座日期范围

## 技术实现

星座分析助手 采用最新的 iOS 开发技术，确保应用的流畅性和准确性：

- 使用 SwiftUI 构建现代化的用户界面
- 集成 Swiss Ephemeris 库提供准确的星体位置计算
- 遵循 Apple 最新的设计规范和隐私标准
- 优化电池使用和性能表现
- 支持深色模式和动态字体

### 第三方库

- **Swiss Ephemeris**: 高精度天文历法计算库
  - GitHub: https://github.com/astrorigin/swisseph
  - 提供精确的行星位置、星座计算等天文数据
