# 复制星座

## 应用简介

复制星座 是一款专业的星座信息复制工具，核心功能是将星座信息转换为可以让大模型分析的文字格式。我们致力于为用户提供便捷的星座信息复制服务，让大模型能够更好地理解和分析星座相关内容。

## 主要功能

### 1. 星座信息复制
- 复制基本星座信息
- 复制星座特征描述
- 复制星座日期范围
- 生成适合大模型分析的文本格式

## 技术实现

复制星座 采用最新的 iOS 开发技术，确保应用的流畅性和准确性：

- 使用 SwiftUI 构建现代化的用户界面
- 集成 Swiss Ephemeris 库提供准确的星体位置计算
- 遵循 Apple 最新的设计规范和隐私标准
- 优化电池使用和性能表现
- 支持深色模式和动态字体

### 第三方库

- **Swiss Ephemeris**: 高精度天文历法计算库
  - GitHub: https://github.com/astrorigin/swisseph
  - 提供精确的行星位置、星座计算等天文数据
